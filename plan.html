💰 Profit Share Model — No Platform Deposits
Solana Mode: User connects Phantom → delegates trade authority to agent → agent trades on user's account → on-chain settle_epoch() auto-splits 20% profit to platform wallet

CEX Mode: User pastes trade-only API keys → agent trades on user's exchange account → per-trade P&L tracked → monthly invoice for 20% profit share

Key: Money NEVER enters our platform. It stays in the user's own wallet/exchange. We only get paid from profits, automatically (Solana) or invoiced (CEX).
